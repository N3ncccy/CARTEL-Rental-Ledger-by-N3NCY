## 📖 О проекте

**CARTEL** — это лёгкое веб-приложение для тех, кто сдаёт машины в аренду на серверах **GTA 5 RP**.

Больше не нужно вести подсчёты в блокноте или в личных сообщениях — просто заводишь автомобиль один раз, выставляешь цену за час и срок аренды, и приложение само:

- 🧮 считает, сколько ты заработаешь с каждого лота;
- 📊 показывает аналитику дохода за день / неделю / месяц / всё время;
- 🚗 хранит статистику по **каждой отдельной машине**;
- 💾 сохраняет всё локально в браузере — данные никуда не улетают и не пропадают между сессиями.

Никаких серверов, регистраций и интернета — открыл файл и работаешь.

## ✨ Возможности

| | |
|---|---|
| 🚘 **Пресеты автомобилей** | Один раз добавь машину с фото, названием и категорией — дальше просто выбирай её из списка |
| 🎚️ **Гибкая сдача в аренду** | Регулируй цену за час и срок аренды (от 1 до 99 часов) ползунком прямо перед сдачей |
| 🧮 **Живой калькулятор дохода** | Итоговая сумма аренды считается на лету, ещё до подтверждения |
| 📦 **Карточки активных лотов** | Фото, статус «В аренде / Готов к возврату», прогресс-бар и обратный отсчёт по каждой машине |
| 📊 **Общая аналитика** | Доход за сегодня, 7 дней, 31 день и за всё время + график по дням |
| 🔎 **Аналитика по каждой машине** | Клик по автомобилю — и видно, сколько именно он принёс и в какие дни |
| 🪪 **Личный кабинет** | Свой ник, аватар и статистика — сколько всего машин сдано в аренду |
| 🌗 **Светлая и тёмная тема** | Переключение в один клик, приятно смотреть и днём, и ночью |
| 🇷🇺 **Полностью на русском** | Интерфейс адаптирован под русскоязычное RP-сообщество |
| 📴 **Работает офлайн** | Один HTML-файл, никаких серверов и установок — все данные хранятся локально |
| 🖼️ **Загрузка фото** | Перетащи изображение, вставь из буфера обмена или выбери файл — как удобно |

## 🚀 Быстрый старт

Никакой установки не требуется — приложение представляет собой один самодостаточный HTML-файл.

```bash
# 1. Клонируй репозиторий
git clone https://github.com/<ваш-профиль>/CARTEL-Rental-Ledger-by-N3NCY.git

# 2. Перейди в папку проекта
cd CARTEL-Rental-Ledger-by-N3NCY

# 3. Просто открой файл в браузере
#    (двойной клик по файлу или перетащи его в окно браузера)
```

Либо ещё проще — скачай `.html`-файл из репозитория и открой его в Chrome, Edge или любом современном браузере. Готово, можно работать.

> 💡 **Совет:** данные хранятся в `localStorage` конкретного браузера. Если хочешь пользоваться на нескольких устройствах — держи файл в одном месте (например, в облаке) и открывай его оттуда.

## 🛠️ Технологии

- 🟨 **Vanilla JavaScript** — без фреймворков и сборщиков
- 🎨 **HTML5 + CSS3** — с переменными для светлой/тёмной темы
- 💾 **localStorage API** — вся база данных живёт прямо в браузере
- 🔤 **Google Fonts** — Sora, Inter, JetBrains Mono

Ни одной серверной части, ни одной внешней базы данных — всё максимально просто и прозрачно.
